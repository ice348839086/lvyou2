**场景**: 用户想找适合亲子游的免费景点

```

- 类型: 全部

- 人群: 亲子家庭 ✓
- 天气: 全部

```

### 2. AI个性化推荐

**输入:**
```
- 目的地: 北京
- 天数: 3天
- 人员: 2大人 + 1小孩
- 节奏: 休闲
- 兴趣: 历史文化
```

**AI输出:**
```
方案A: 深度体验版
- Day1: 故宫(3小时) + 景山(1.5小时)
- Day2: 长城(4小时)
- Day3: 天坛(2小时) + 前门(2小时)

理由:
- 考虑有小孩,每天安排2-3个景点
- 选择历史文化类景点
- 避开过长步行路线
- 预留充足休息时间
```

### 3. 实时地图交互

**操作流程:**
```

   ↓
5. 用户看到实时更新的结果
```

## 💡 设计亮点

### 1. 渐变色彩系统

- **主色调**: 蓝紫色渐变 (from-blue-600 to-purple-600)
- **背景**: 多层渐变营造空间感
- **卡片**: 白色卡片+阴影提升层次

### 2. 图标系统

- 🏛️ 历史遗迹
- 🌳 自然风光
- 🎨 文化艺术
- 🏙️ 现代建筑
- 🍜 美食体验
- 🛍️ 购物休闲

### 3. 动画效果

- **淡入动画**: 页面切换时的fade-in
- **加载动画**: 旋转的圆环
- **进度条**: 平滑的宽度变化
- **悬停效果**: 卡片阴影增强

### 4. 响应式设计

- **桌面端**: 双列布局,充分利用空间
- **平板端**: 单列布局,保持可读性
- **移动端**: 全屏显示,触摸优化

## 🎬 使用场景

### 场景1: 周末短途游

**用户**: 上海白领,想周末去杭州
**操作**:
1. 选择目的地"杭州"
2. 设置2天1夜
3. 选择"情侣约会"标签
4. AI生成浪漫路线

**结果**: 西湖+武康路+河坊街的文艺路线

### 场景2: 家庭春节游

**用户**: 一家四口(2大人+2小孩)
**操作**:
1. 选择"北京"
2. 设置5天4夜
3. 选择"休闲"节奏
4. 勾选"亲子友好"

**结果**: 避开爬山景点,增加休息时间,选择适合小孩的景点

### 场景3: 深度文化游

**用户**: 历史爱好者
**操作**:
1. 筛选"历史遗迹"类型
2. 选择"深度体验"时长
3. 查看AI摘要的"隐藏玩法"

**结果**: 发现小众博物馆和历史街区
